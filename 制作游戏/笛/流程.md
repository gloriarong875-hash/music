创建一个仿照《千年调》博物馆交互风格的网页小游戏，主题为"笛韵千年——中国传统竹笛制作"。

## 视觉规范
- 背景色：深炭灰 #1a1a1a，无多余装饰
- 强调色：古铜金 #c9a96e，暗褐 #5a4a3a
- 中央交互对象：国画风格
- 信息按钮：两侧圆形悬浮按钮，鼠标悬停显示提示，点击弹出文化介绍弹窗


## 四步流程（真实还原传统制笛）

第一步【择竹】
- 竹林场景（图片，可更改），展示3根不同竹材（图片可更改）通过鼠标滚动放大背景，移动三种竹子素材的视差滚动模拟竹林前进后退
- 交互：鼠标悬停显示竹龄、直径等信息；
- 视觉反馈：可选中竹材泛白光，前两个不可选中，悬停时变灰
- 文化注释："冬至伐竹，竹龄三年，壁厚均匀者为上"
    - 第一根竹子为    ○ 嫩竹（文字浮现：应取3年以上的老竹，直径2-3公分
    - 第二根竹子为○ 歪竹（文字浮现：应取质地均匀、壁厚杆直、无虫蛀的竹材）
-最后选中● 自动播放：新竹阴干（竹子变色，废料被剔除）
新竹砍下需要阴干2年以上

第三步【开孔】
- 中央展示已截取的竹管，表面有墨线标记
- 交互：依次点击吹孔、膜孔、六个音孔位置
- 每开一个孔：木屑粒子效果，孔洞边缘焦褐（模拟烧烙工艺）
- 辅助出现：古代律管尺度图半透明叠加
- 文化注释："依十二律开孔，吹孔为宫，膜孔应商，六孔配五声二变"


第四步【试音】
- 中央展示笛膜材料选项（芦苇膜、竹膜、纸膜实物特写）
- 交互：拖拽选中膜材至膜孔位置
- 视觉：膜材自动拉伸贴合，出现细微震颤波纹
- 三种膜材影响最终音色（芦苇清亮/竹膜温润/纸膜浑厚）
- 文化注释："笛膜为魂，采芦苇内膜，薄如蝉翼，透似轻纱"
- 中央展示完成品笛，七孔俱全
- 交互：点击各孔或按键盘数字键1-7发声
- 视觉：音孔发出同心圆光波，右侧浮现工尺谱字

- 终局：笛身泛起柔和光晕，底部出现"再制一管"和向下箭头两个圆形按钮
- 文化注释："笛开七孔，上应七星。五声二变，清浊自辨"

## 素材要求
- 竹材：浙江余杭苦竹/江西蕲竹实物高清图
- 古笛参考：贾湖骨笛（河南博物院）、曾侯乙竹笛
- 工具：参照《考工记》古法刀具样式
- 人物：不使用卡通角色，如需人物则采用古画局部抠图（如《韩熙载夜宴图》吹笛乐工）

## 音效设计
- 择竹：竹节轻叩声（Web Audio API合成）
- 开孔：烙铁灼竹的轻微爆裂声
- 贴膜：薄膜震颤的细微"嗡"声
- 试音：真实竹笛五声音阶采样，带自然混响

## 技术约束
- 单文件HTML，内嵌CSS/JS
- 图片使用博物馆公开资源URL占位
- 响应式布局，支持触屏
- 所有动画使用CSS transition，避免复杂JS动画库
Output format: Complete, self-contained HTML file with:
- All CSS in <style> tag
- All JS in <script> tag  
- Image URLs using placeholder services or museum open data
- Ready to save as .html and open in browser
